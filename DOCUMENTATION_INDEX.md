## 🔍 Quick Find

### Commands

**Start Local:**
```bash
start-local.bat
start-backend.bat
start-frontend.bat
```

**Deploy AWS:**
```bash
cd infra/terraform
terraform init
terraform apply
```

**Check Costs:**
```bash
cd infra/scripts
./cost-optimization.sh costs
```

### URLs

**Local Frontend:** http://localhost:3000
**Local Backend:** http://localhost:4000
**API Health:** http://localhost:4000/health
**RabbitMQ UI:** http://localhost:15672

### Key Files

**Environment:** `.env`, `.env.example`
**Config:** `docker-compose.yml`, `package.json`
**Terraform:** `infra/terraform/*.tf`
**Scripts:** `infra/scripts/*.sh`, `*.bat`
